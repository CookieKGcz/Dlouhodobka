# Usability and usable security
## Usefulness summary
- = Utility (effectiveness) + Usability (learnability, efficiency, memorability, errors, satisfaction)
- Context!
- Specific for sec:
	- Protect, not obscure
### Utility
- Effectiveness
	- Does the system do what users need?
	- Does the user succeed?
- Usability
	- Learnability
		- How easy is it for users to succeed the first time they encounter the system?
		- VIM x General IDE
	- Efficiency
		- Having learned the system, how quickly can users preform tasks?
	- Memorability
		- Returning after some time, how easily can users re-establish proficiency?
	- Errors
		- How many errors do users make?
		- How severe are they?
		- How easy can they recover from them?
	- Satisfaction
		- How pleasant is it to use the system?
## Usable security
- is a multidisciplinary field investigating and improving system security (and privacy) focusing on human interaction with the system
