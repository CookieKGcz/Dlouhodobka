## Datové typy a proměnné
### Proměnná a datový typ
- **Proměnná je pojmenované uložiště pro hodnotu určitého datového typu.**
	- má tedy jméno a typ hodnot, které může mít
	- může být vytvořena a zrušena
	- Po dobu existence hodnota zabírá určitou paměť v počítači
- **Datový typ vymezuje množinu hodnot pro proměnné.**
- **Hodnota představuje určitou informaci (např. číslo – teplota, tlak,…)**
	- může bát vytvořena a zrušena
	- Po dobu existence hodnota zabírá určitou paměť v počítači

- Dva způsoby uložení hodnoty v proměnné:
	- **Hodnotou**: Hodnota je uložena přímo v paměti proměnné
	- **Odkazem**: V paměti proměnné je uložen pouze odkaz (adresa) na místo v paměti, kde je uložena hodnota.

### Třídy
- vytváření nových datových typů
- vymezuje množinu hodnot

- Třída určuje, kolik proměnných kterých typů se v každé instanci třídy nachází
	- = atributy nebo položky

- V Javě můžeme třídu definovat takto: ```

```

## Operátor, výraz a příkaz
### Operátor, výraz a příkaz
- **Operátor** je symbol označující výpočet, který jazyk poskytuje
- Java poskytuje celou řadu operátorů, např.: 
	- Aritmetické: +, -, *, /, % (zbytek po dělení), … 
	- Relační: == (rovnost), != (nerovnost), <, <=, >, >=. 
	- Logické: && (konjunkce), || (disjunkce), ! (negace) 
	- Přiřazení: =, += (zkratka za a=a+…), … 
	- Přetypování: (), např. (int)3.14 se vyhodnotí na číslo 3 typu int.
- **Výraz** je část programu, která nám vypočte určitou hodnotu.

- Příklady výrazů: Uvažujme proměnné int i, int j, float f s hodnotami po řadě 3,-2, 1.5f. Pak výrazy jsou např.:
	- 2\*i = 6
	- (float)j+f = -0.5f
	- i < j = false
	- i == (int)(2\*f) && i-j != 2   = true

- `class A { int i; float f; } class B { A a; B b; }`
- ![[Pasted image 20260218090740.png | 300]]
- x.i vrátí 42
- y.a.f vrátí 3.14
- y.b.b.a.i vrátí 42
- x.f <= 2.3 vrátí false
- y.a.i + x.i == 2\*y.b.a.i vrátí true
### Přiřazení do proměnných
- **Příkaz** je část programu jehož vypočet nám hodnotu nedá.
- Příkazem přiřazení vkládáme hodnoty do proměnných :
	- <výraz-určující-proměnnou> = <hodnota-nebo-výraz>;
	- int i; char c; A a;
		- i = 5; c = ‘A’; a = new A();
		- nebo přímo v definici
			- int i = 5; char c = ‘A’; A a = new A();
### Operátor, výraz a příkaz
- Přiřazovat můžeme i do atributů v instancích tříd.
	- Před symbol = napíšeme výraz s operátorem ‘.’.
- `class A { int i; float f; } class B { A a; B b; }`

## Blok, rozsah platnosti, život objektu, metoda
### Blok, rozsah platnosti proměnné
- **Blok** je sekvence žádného, jednoho, nebo více příkazů uzavřených do závorek { a }.
	- vykonávají se v pořadí v jakém jsou uvedeny
	- jde bloky do sebe zanořovat
- ![[Pasted image 20260218092042.png | 450]]
- také `{ int x; { int x; } }` dojde k chybě při překladu
### Život objektu
- Vznik
	- new operátor
- inicializace
	- přiřazení hodnoty jeho proměnným
	- def. hodnoty (0, null,...) se nahradí požadovanými
- používání
	- dokud je objekt dosažitelný
- není dosažitelný -> připraven na odstranění 
	- -> garbage collector

- Objekt může být ve stavu použití i mimo blok, ve kterém byl vytvořen.
	- ![[Pasted image 20260218092525.png | 450]]
### Metoda
- Pojmenovaný blok, který
	- má žádnou, jednu, nebo více vstupních proměnných. Těmto proměnným se říká parametry.
	- vrací nebo nevrací (void) hodnotu
	- při vrácení se použije klíčové slovo return
	- ==metody definujeme uvnitř definice tříd==
	- Každá metoda má implicitní parametr (nepíšeme jej), který se jmenuje this a odkazuje na instanci třídy, v níž je definována.
### Blok a metoda
- Příklad: Do třídy Vector vložte metody (a) nastavující souřadnici x na novou hodnotu, (b) počítající skalární součin.
- `class Vector { float x; float y; }`

- řešení:
```java
class Vector {
	float x; float y;
	
	void setX(float value) { x=value; return; }
	
	float dotProduct() { return x*x + y*y; }
}
```

### Volání funkce
- Metodu můžeme **zavolat** v bloku tak, že ji předáme hodnoty pro všechny její parametry.
- Příklad:
	- Definujte blok, ve kterém vytvořte instanci třídy Vektor z předchozího příkladu, nastavte souřadnici x na hodnotu -1.5f voláním metody setX, a pak získejte hodnotu skalárního součinu.
	- `{ Vector v = new Vector(); v.setX(-1.5f); float d = v.dotProduct(); }`

- Při vykonávání příkazů v blocích a při volání metod hrají důležitou roli dvě oblasti paměti: ==Zásobník== a ==Halda==.
- Při volání metody se na vrchol zásobníku uloží:
	- proměnné definované v blocích metody (lokální proměnné)
	- parametry metody (včetně this)
	- proměnnou pro návratovou hodnotu, pokud metoda vrací hodnotu
- Při návratu se uvedená data zase odstraní
- Na haldu se ukládají všechny vytvořené objekty
